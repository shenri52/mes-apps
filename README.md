# 📲 Mon Portail d'Applications Streamlit

Ce projet est une interface centralisée (Dashboard) permettant d'accéder rapidement à mes différentes applications Streamlit via un seul raccourci sur mobile.

## 🚀 Fonctionnalités
- **Accès Centralisé** : Un seul lien pour piloter toutes vos applications.
- **Design Mobile-First** : Boutons larges, épurés (style contour rouge) et centrés pour une utilisation au pouce.
- **Navigation Directe** : Redirection vers les applications cibles en un seul clic.
